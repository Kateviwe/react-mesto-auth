# __Проект: Место__
---
## Данный проект представляет собой сервис, в котором можно делиться фотографиями, удалять их и ставить лайки. Реализована регистрация и авторизация на сайте.

__В проекте применяются такие технологии:__ 
* Реализована адаптивная верстка
* Используются медиазапросы
* Достигается 'резиновый' эффект
* Создана файловая структура по БЭМ (Nested)
* Применяются Flex и Grid Layout
* Устанавливаются эффекты перехода через CSS
* React
* React Router
* Создание проекта на Create React App
* Работа с 2 API
* Применение HOC-компонента
* Работа с localStorage
